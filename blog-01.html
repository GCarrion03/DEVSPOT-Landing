My journey to become an AWS Certified Solutions Architect Associate

Passed the AWS Certified Solutions Architect Associate (SAA-C01 926/1000) On a budget - Spent 1 month and $20 on acloud.guru AWSCSAA + AWSCDA bundle + $150 on exam

Yesterday morning I passed the test. I studied for 4 weeks starting May 1st. The first 15 days followed the acloud.guru guide for 1 hour daily, the remaining 15 days did multiple free mock tests.

Materials:

  • acloud.guru AWS Certified Solutions Architect - Associate
  • AWS ReInvent videos
  • Free internet resources ( learnersreference.com, quizlet.com, flashcards.io, AWS sample exam, examtopics.com, found a few mock tests on WordPress sites ).

Preparation:

I am a Java / Javascript developer (7Y exp), a year ago started to deploy things on AWS cloud so I was curious how I would do on this exam, this was my plan:

  • Week 1-2: Take the acloud.guru course
  • Week 3: do the in-course acloud.guru mock tests and start looking for resources around the web
  • Week 4: Keep doing mock tests watch Re-Invent videos for IRL use cases.

The test:

Is somewhat biased to use Amazon homegrown products: example always pick Aurora instead of RDS, among other things, also I noted that homegrown AWS services have some of the most popular components enhancements built-in, like caching in dynamoDB DAX vs having to configure ElastiCache for a vanilla RDS so it is worth reviewing the advantages of each one.

Finally I'd like to give something back to the aws community by providing AWS-CSAA aspirants a totally free exam simulator here.

Good luck!
